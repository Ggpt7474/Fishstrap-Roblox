# Fishstrap Roblox

**Fishstrap Roblox** is a powerful custom launcher for Roblox. It provides advanced FastFlags editing, multi-instance support, performance tweaks, and better control over your Roblox experience compared to the official launcher.

## Key Features
- Advanced FastFlags editor with hidden options
- Multi-instance support (run multiple Roblox clients)
- Custom Roblox version management
- Performance optimizations and lag reduction
- Clean, modern user interface
- Automatic updates for Roblox and launcher

## Getting Started
1. **Download** the latest version using the button above.
2. **Extract** the archive.
3. **Run** `Fishstrap.exe` as Administrator.
4. **Configure** your FastFlags and settings.
5. **Launch Roblox** through the launcher.

**Pro Tip:** Use the FastFlags editor to unlock higher FPS and better performance.

## System Requirements
- **Windows 10 / 11** (64-bit)
