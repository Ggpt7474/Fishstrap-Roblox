## ⚙️ Setting Up Your Experience

The application opens to a settings menu. You can change how your game looks and feels here. 

### Performance Tweaks
You want a smooth frame rate. Navigate to the Performance tab. Enable the FPS unlocker to remove hardware limits on your screen refresh rate. You can also disable heavy textures or unnecessary background noise to save system memory.

### FastFlag Management
FastFlags tell the game engine how to behave. Most users find these settings hard to change. Fishstrap simplifies this. Select "FastFlag Editor" from the sidebar. You can toggle common settings like shadows, lighting quality, and rendering modes without editing system files. 

### Discord Presence
You can show your friends what game you play. Go to the Integrations tab. Turn on Discord Rich Presence. This displays your current session status, map name, and server count on your Discord profile.

## 🛠 Troubleshooting Common Issues

Technology rarely works without occasional bumps. Check these tips if the app stops working.

### Crashing on Startup
If the tool crashes before it opens, ensure you have the latest .NET Runtime installed on your computer. You can find this on the official Microsoft website. Restart your computer after you install these support files to ensure your system recognizes the update.

### Game Not Updating
Sometimes the launcher fails to see new game updates. Open the Fishstrap settings menu and click the "Force Update" button. This forces the software to re-check the game server and pull the latest versions of the required files.

### Runtime Errors
Windows might occasionally block the software if your antivirus software is aggressive. Add an exclusion folder for the Fishstrap directory in your antivirus settings. This tells your computer to trust the launcher files.
